# Using GNS3 to understand OSPFv2 & RIPv2 protocols

This assignment deals with simulating networks to understand the functioning of Routing Information Protocol (RIP) & Open Shortest Path First (OSPF) using [GNS3](https://www.gns3.com/)

[Download GNS3](https://www.gns3.com/software/download)

`c3640` Cisco Router was used, image for which can be found on net, otherwise any other router can easily be replaced

All the projects are in zip files, with some Wireshark captures included. `extras` folder contains commands run on each router and PC, in the order as they appear in the assignment pdf
